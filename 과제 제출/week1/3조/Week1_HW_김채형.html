변수명 변수 설명 값
instant record index -
dteday date yyyy-mm-dd
season season 1:spring / 2:summer / 3:fall / 4:winter
yr year 0: 2011 / 1:2012
mnth month 1~12
hr hour 0 to 23
holiday - weather day is holiday or not
weekday - day of the week
workingday - if day is neither weekend nor holiday is 1, otherwise is 0
weathersit - 1: Clear, Few clouds, Partly cloudy, Partly cloudy / 2: Mist + Cloudy, Mist + Broken clouds, Mist + Few clouds, Mist / 3: Light Snow, Light Rain + Thunderstorm + Scattered clouds, Light Rain + Scattered clouds / 4: Heavy Rain + Ice Pallets + Thunderstorm + Mist, Snow + Fog
temp Normalized temperature The values are derived via (t-tmin)/(tmax-tmin), tmin=-8, t_max=+39 (only in hourly scale)
atemp Normalized feeling temperature The values are derived via (t-tmin)/(tmax-tmin), tmin=-16, t_max=+50 (only in hourly scale)
hum Normalized humidity The values are divided to 100 (max)
windspeed Normalized wind speed The values are divided to 67 (max)
casual count of casual users
registered count of registered users
cnt count of total rental bikes including both casual and registered

  • casual, registered, cnt
    아마도 casual과 registered 합치면 cnt 되는 것 같음, 그러면 cnt만 남기자

train_mse=[]
test_mse=[]
for i in np.arange(2,13):
    poly = PolynomialFeatures(degree=i)
    X_train_poly = poly.fit_transform(X_train)
    X_test_poly = poly.fit_transform(X_test)
    
    linreg=LinearRegression(fit_intercept=False)
    linreg.fit(X_train_poly, Y_train)
    
    train_prediction=linreg.predict(X_train_poly)
    tr_mse = math.log(sum(np.array(train_prediction-Y_train)**2)/len(X_train))
    train_mse.append(tr_mse)
    
    test_prediction=linreg.predict(X_test_poly)
    te_mse = math.log(sum(np.array(test_prediction-Y_test)**2)/len(X_test))
    test_mse.append(te_mse)
print(train_mse)    
print(test_mse)

plt.plot(np.arange(2,13), train_mse, marker='o')
plt.plot(np.arange(2,13), test_mse, marker='o')
